# Gist Java SDK

Implement a java sdk (using jdk 17 httpclient) refer to GitHub official doc:
https://docs.github.com/en/rest/gists?apiVersion=2026-03-10

Implement the below apis:

- List gists for the authenticated user
- Create a gist
- List public gists
- List starred gists
- Get a gist
- Update a gist
- Delete a gist
